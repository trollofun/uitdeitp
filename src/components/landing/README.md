# Landing Page Components

Production-ready React components following Gestalt design principles.

## Components

### HeroSection
Focal hero section with single point of attention, symmetric layout, and clear CTAs.

**File:** `HeroSection.tsx`
**Size:** ~5 KB
**Principles:** Pragnanz, Symmetry, Figure/Ground, Proximity

### FeaturesGrid
Three identical feature cards demonstrating Gestalt Similarity principle.

**File:** `FeaturesGrid.tsx`
**Size:** ~4 KB
**Principles:** Similarity, Proximity, Symmetry, Figure/Ground

## Quick Start

```tsx
import { HeroSection, FeaturesGrid } from '@/components/landing';

<HeroSection />
<FeaturesGrid />
```
