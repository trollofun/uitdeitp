# PWA Quick Start Guide

**5-Minute Guide to Install uitdeITP Kiosk on Tablets**

---

## Android Installation (Chrome/Edge)

### Method 1: Automatic Prompt (Easiest)

```
1. Open kiosk URL: https://uitdeitp.ro/kiosk/{station-slug}
2. Wait 3 seconds
3. Tap "Instalează Acum" banner
4. Confirm installation
5. Launch from home screen
```

### Method 2: Browser Menu

```
1. Visit kiosk URL
2. Tap ⋮ (3 dots menu)
3. Select "Add to Home screen" or "Install app"
4. Confirm
5. Launch from home screen
```

---

## iOS Installation (Safari)

**Only Method:**

```
1. Open kiosk URL in Safari
2. Tap Share button (□↗)
3. Scroll and select "Add to Home Screen"
4. Tap "Add"
5. Launch from home screen
```

⚠️ **Must use Safari** (not Chrome/Edge on iOS)

---

## Verify Installation

✅ **Icon on home screen** with station logo
✅ **Launches fullscreen** (no browser UI)
✅ **Kiosk flow works** (name → phone → plate → date)

---

## Troubleshooting

**Install prompt doesn't show (Android):**
- Check if already installed (look for icon)
- Try browser menu method
- Update Chrome/Edge to latest version

**Can't find "Add to Home Screen" (iOS):**
- Make sure using Safari (not Chrome)
- Update iOS to latest version
- Check Screen Time restrictions

**Not fullscreen after install:**
- Uninstall and reinstall
- Make sure installed (not bookmarked)
- Verify correct URL (not homepage)
